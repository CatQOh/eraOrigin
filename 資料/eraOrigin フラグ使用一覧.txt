【eraBEMANI フラグ使用状況一覧】
FLAG
　FLAG:0 = 休憩フラグ
　FLAG:1 = 初期着衣の設定(0=全着 1=下着無し 2=下着 3=全裸)
　FLAG:6 = 調教時テキストの表示/非表示
　FLAG:7 = キャラ口上の表示/非表示
　FLAG:8 = コンドームの自動装着処理
　　ビット演算により処理
　　0,1 = 主人
　　1,2 = 助手
　　2,4 = 奴隷
　FLAG:9 = SHOPからABL上昇処理に飛んだ場合のターン終了回避
　FLAG:11 = 気温(値は気温×10)
　FLAG:12 = 曜日(1:日,2:月,3:火,4:水,5:木,6:金,0:土)
　FLAG:13 = 天気(1=晴れ 2=曇り 3=雨 4=大雨 5=雷雨 6=雪 7=大雪)
　FLAG:14 = 降水量
　FLAG:15 = 梅雨フラグ

　FLAG:20 = 恋慕をつけた人数
　FLAG:21 = 淫乱をつけた人数
　FLAG:22 = 服従をつけた人数

　FLAG:40 = 色仕掛けイベントのオン/オフ

　TFLAG:121 = 吸血済み判定フラグ
　TFLAG:122 = 腸洗浄済み判定フラグ
